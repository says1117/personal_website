# Old Gods — exact-image drop folder

The About page auto-detects these files. Drop them in with these EXACT
names and hard-refresh — no code changes needed. Remove a file and the
page falls back to the original vector art for that element.

| File              | Used as                                        |
| ----------------- | ---------------------------------------------- |
| `archmother.jpg`  | Archmother realm full-bleed background (blue)  |
| `hiddenking.jpg`  | Hidden King realm full-bleed background        |
| `soul.png`        | The playable soul-orb skin (circle-cropped)    |
| `calico.png`      | Calico the cat (transparent-background PNG     |
|                   | recommended — remove the backdrop for best     |
|                   | results; a plain screenshot will show its box) |

⚠ These are Valve's copyrighted game images. Keep this build
private/personal — do NOT deploy these files to the public site. The
repo's vector art is the publishable version, and the site credits
state that no Valve artwork is used.
